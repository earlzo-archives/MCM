初步浏览
First galance
题目C提供的数据包括了2977所候选学校,而记分牌数据有7804条学校数据,因此我们第一步需要做的是将候选名单中的学校数据挑选出来,我们通过唯一的机构ID来对数据进行筛选,发现有41所学校是没有数据的,为了填补这些缺失数据,我们决定IPEDS收集这些学校的数据.

数据收集
Data collection
IPEDS为我们提供了详尽的各类报告和统计数据并区分了年份,统计变量等,但是这反而增加了我们筛选出合适的难度.我们干脆下载了IPEDS的所有数据.

文件检验
file validation
将IPEDS所有的数据下载并解压后,共有913个数据记录文件,我们仍然通过机构ID来检验数据的完整性,当数据不包含所有候选学校时认为数据是缺失的.我们编写了一个脚本完成这项工作并导出报告.

表格 IPEDS数据完整性报告
缺失  错误  完整  合计
903   7     3     913

我们分析了数据完整的3个数据文件,但是并没有记录能有效衡量学校状态的数据,最终我们决定将缺失数据的学校排除.

数据提取
data extraction
我们提取了候选名单中拥有数据的学校的所有记分牌数据,共28936条.同时我们也导出了记分牌数据变量字典以理解数据.记分牌数据变量中有一个表示当前学校是否在运转的变量'CURROPER',进一步的,我们将不再运转的学校进行排除,最终实际参与到排名的学校剩下2936所.

字段分离
Variable separation
记分牌包括一部分描述了学校的地址,网站,ID,位置,机构类型等不可计算的数据,我们仅将能够参与计算数据分离出来,包括了描述了学校招生状况,学生毕业状况以及学生支出,就业后工作收入等98个变量.

缺失值处理
Handling Missing Data
虽然得到了所有的数值型变量数据,但是里面存在了大量的缺失数据（当某字段值为"NULL"或者"PrivacySuppressed"便为缺失）,通过分析发现, 大部分缺失值是由于机构类型不同而造成的,一个机构类型分别由是"公立机构"还是"私立机构"和是"四年制机构"还是"小于四年制机构"确定,由于涉及到公立,私立机构类型的变量为不同家庭状况下的平均家庭支出不会参与到我们随后的计算,我们直接舍去了这部分数据.
接下来我们还剩下区分机构年制的六个变量,我们通过这六个变量是否有值来对机构进行分类,最终,我们得到了四年制机构数据 1896 条, 小于四年制机构数据 1007 条, 未知年制机构数据 25 条, 同样的,我们对缺失未知年制机构数据进行舍去,而对于分好类的数据的缺失数据,我们将缺失数据填充为该变量的平均值.
